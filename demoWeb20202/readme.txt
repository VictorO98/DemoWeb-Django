Aportes del framework
    * Patrón arquitectonico MVT : URLConf -> Vista -> Modelo (-> BD o Res...) -> Plantilla -> Navegador Web
    * Creación de módulos con facilidad : python .\manage.py startapp <nombreModulo>
    * Nos ayuda a crear una base de datos de prueba por defecto, ahorra tiempo 
        en gestionar una base de datos: python manage.py migrate
    * Nos crea un super usuario automaticamente: python manage.py createsuperuser
